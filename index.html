設定 Apify Token

本工具需要 Apify API Token 來抓取 Threads 公開資料。Token 僅儲存在你的瀏覽器本機，不會上傳至任何伺服器。

免費申請：apify.com → Settings → Integrations → API tokens
Token 儲存於瀏覽器 localStorage，關閉頁面後仍有效。
